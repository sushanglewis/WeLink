# 功能清单：Lincoln TUI 录音工具

**设计 ID**: checkout-redesign

---

## 功能列表

| 优先级 | 功能 | 描述 | 验收标准 | 来源 |
|---|---|---|---|---|
| P0 | 全局 CLI 入口 | `npm install -g lincoln` 后，`lincoln` 命令可用 | 任意目录输入 `lincoln` 可启动 | 需求文档 |
| P0 | TUI 录音界面 | 使用 `ink` 展示动态录音状态 | 显示脉冲效果、时长、停止提示 | `[00:00:19 - 00:00:26]` |
| P0 | 开始录音 | 启动 TUI 后立即开始录制麦克风音频 | 3 秒内进入录音状态 | `[00:00:02 - 00:00:19]` |
| P0 | 停止并保存 | 按 `Enter` 停止录音并保存产物 | 产物保存到 `recordings/` 和 `interviews/` | `[00:00:37 - 00:00:52]` |
| P0 | 取消录音 | 按 `q` 或 `Ctrl+C` 取消，不保存 | 无产物生成，优雅退出 | 用户澄清 |
| P0 | 复用现有后端 | 调用 Python `record-interview` CLI 录音 | 产物格式与 CLI 一致 | 用户澄清 |
| P1 | 自动 session_id | 从日期、git 分支、工作区名派生 | 无需手动输入 session_id | 需求文档 |
| P1 | 处理确认 | 停止后询问是否触发 `process-interview` | 用户可选择立即/稍后处理 | 需求文档 |
| P0 | 音频电平条 | 在 TUI 中展示实时音频电平 | 电平随麦克风输入实时变化 | 用户澄清 |
| P0 | 全局配置 | 支持 `~/.lincolnrc` 保存默认参数 | 配置可被命令行参数覆盖 | 用户澄清 |

## 验收映射

| 需求来源 | 覆盖功能 |
|---|---|
| 触发方式不友好 `[00:00:02 - 00:00:19]` | 全局 CLI 入口、自动 session_id |
| 录音状态不可见 `[00:00:19 - 00:00:26]` | TUI 录音界面 |
| 集成位置不自然 `[00:00:26 - 00:00:37]` | 终端原生、跨 AI 工具 |
| 交互过于高调 `[00:00:52 - 00:00:55]` | TUI 占用当前终端标签、柔和动态效果 |
| 实时反馈需求 | 音频电平条 |

## 非功能需求

- **性能**：TUI 启动到开始录音不超过 3 秒。
- **兼容性**：macOS 优先；产物格式与现有 CLI 一致。
- **可维护性**：TUI 层与录音后端解耦，后端升级不影响前端。
