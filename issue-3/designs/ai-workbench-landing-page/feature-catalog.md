# Feature Catalog: 企业 AI 工作台一级落地页

## P0 功能清单与验收标准

### FR-001：AI 工作台一级入口

| 编号 | 功能 | 验收标准 |
|------|------|----------|
| FR-001-1 | WeLink 承载 AI 工作台壳层 | 登录后用户可在 WeLink 内进入 AI 工作台，壳层提供最小导航 |
| FR-001-2 | 复用 LibreChat 前端页面 | AI 工作台主界面直接展示 LibreChat 页面，不重复绘制 UI/UX |

**需求来源**：[requirements.md#方案](../../requirements/github-issue-3/requirements.md)

### FR-002：AI 助手栏 / 多轮对话

| 编号 | 功能 | 验收标准 |
|------|------|----------|
| FR-002-1 | 自然语言输入 | 用户可在输入框输入文字，按回车发送 |
| FR-002-2 | 多轮对话 | AI 可基于上下文连续对话 |
| FR-002-3 | 企业 model config | 对话调用企业统一提供的模型配置 |
| FR-002-4 | 复用 LibreChat 原生对话 UI | 不重新设计对话界面 |

### FR-003：知识库问答

| 编号 | 功能 | 验收标准 |
|------|------|----------|
| FR-003-1 | 文档上传 | 支持上传 PDF、Word、Markdown 等文档 |
| FR-003-2 | 向量化 | 使用企业提供 embed 模型构建索引 |
| FR-003-3 | RAG 问答 | 基于上传文档回答用户问题 |
| FR-003-4 | 答案溯源 | 答案展示引用文档或段落 |
| FR-003-5 | 复用 LibreChat 原生知识库能力 | 不重新设计知识库 UI |

### FR-004：Canvas / Artifacts

| 编号 | 功能 | 验收标准 |
|------|------|----------|
| FR-004-1 | 生成内容 | 支持生成 Markdown、HTML、Mermaid、SVG 等 artifacts |
| FR-004-2 | 在线编辑 | 用户可在 Artifact 窗口中编辑内容 |
| FR-004-3 | 下载导出 | 用户可下载生成的内容 |
| FR-004-4 | 暂不支持 PPT | 若 LibreChat 不支持 PPT，本期不实现 |
| FR-004-5 | 复用 LibreChat 原生 Artifacts | 不重新设计 Canvas UI |

### FR-005：个人级使用数据收集

| 编号 | 功能 | 验收标准 |
|------|------|----------|
| FR-005-1 | 会话记录 | 记录用户每次会话内容 |
| FR-005-2 | 功能调用记录 | 记录用户调用的 LibreChat 功能（知识库、Artifacts 等） |
| FR-005-3 | Canvas 使用记录 | 记录用户生成的 artifacts 类型和次数 |
| FR-005-4 | 数据落库 | 数据写入企业后台数据库 |
| FR-005-5 | 基础查询 | 支持按用户/时间/功能维度查询 |

### FR-006：规划中功能（本期不做）

| 编号 | 功能 | 说明 |
|------|------|------|
| FR-006-1 | 应用网格设计 | 列入后续规划，本期最大化复用 LibreChat 原生能力 |
| FR-006-2 | 督办场景集成 | 依赖 `designs/supervision-collab-va` 功能上线后再接入 |
| FR-006-3 | 管理员 skill 发布 | 企业自定义 skill 配置后台，后续规划 |
