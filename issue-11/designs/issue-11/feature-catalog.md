## 非功能需求

- **性能**：应用启动时间 ≤ 3 秒（普通办公电脑）。
- **稳定性**：支持 7×24 小时驻留，崩溃后可选自动重启。
- **安全性**：服务器 URL / 认证配置打包时混淆或加密；token 加密存储。
- **可维护性**：清晰划分 Shell / Bridge / WebView / Settings 模块。
- **兼容性**：Windows 10+ / macOS 12+；x64 + Apple Silicon。
