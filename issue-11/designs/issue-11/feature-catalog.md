# 功能目录：issue-11 企业 IM 桌面应用

## 功能清单

| 功能 | 描述 | 优先级 | 验收标准 | 来源 |
|------|------|:------:|----------|------|
| 安装包预配置 | 安装包内置服务器 URL | P0 | 启动后无 URL 输入入口 | 需求/用户故事一 |
| 应用级统一登录 | 启动后进入 EAIC 引导程序，使用已同步的企业邮箱/密码登录 | P0 | 登录成功并进入主界面 | 需求/用户故事二 |
| 原生一级导航 | 通讯录、聊天、AI 表格以 C/S 左侧边栏导航呈现 | P0 | 登录后展示并支持切换 | 需求/用户故事三 |
| WebView 嵌入聊天 | 聊天区域加载现有 B/S 页面 | P0 | 消息收发正常 | 需求/用户故事三 |
| 通知统一 | WebView 未读/红点通过 JS Bridge 同步到 C/S 托盘与系统通知 | P0 | 托盘红点与 B/S 页面红点一致 | 设计确认 |
| 系统通知 | 新消息系统级弹窗 | P0 | 在线时弹出，点击可唤出 | 需求/用户故事四 |
| 系统托盘 | 关闭窗口后常驻托盘 | P0 | 点击托盘唤出应用 | 需求/用户故事五 |
| 开机自启动 | 可配置开机自启 | P1 | 设置项生效 | 需求 |
| 品牌定制 | 标题、图标、关于页替换 | P0 | 无 Mattermost 暴露 | 需求/用户故事六 |
| 设置适配 | 个人/系统设置重排重命名 | P1 | 符合中国用户习惯 | 需求/用户故事七 |
| 文件下载/打开 | 聊天文件下载并系统打开 | P1 | 下载成功并调用默认应用 | PRD |
| 深度链接 | 外部链接跳转应用内指定会话 | P2 | 链接可正确唤起并定位 | PRD |

## 验收映射

- **P0 功能**：MVP 必须完成，影响上线。
- **P1 功能**：上线后短期内补齐。
- **P2 功能**：后续迭代中实现。

## 非功能需求映射

| 非功能需求 | 设计体现 |
|------------|----------|
| 启动 ≤ 3 秒 | Tauri 轻量启动；预加载首页 WebView |
| 7×24 驻留 | 崩溃监控与可选自动重启；内存泄漏监控 |
| 配置安全 | 服务器 URL / SSO 配置打包时混淆或加密 |
| 可维护性 | 清晰划分 Shell / Bridge / WebView / Settings 模块 |
| 兼容性 | Windows 10+ / macOS 12+；x64 + Apple Silicon |
