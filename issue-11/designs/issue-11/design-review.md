# 设计评审：issue-11 企业 IM 桌面应用

<!-- status: pending -->

## 决策摘要

- **产品**：基于现有 Mattermost B/S 二开系统，打造企业自有品牌的 Windows/macOS 桌面应用。
- **架构**：混合 C/S + B/S。Tauri 构建原生桌面外壳；一级导航（通讯录、聊天、AI 表格）为 C/S；聊天等核心功能初期通过 WebView 嵌入现有 B/S 页面。
- **技术栈**：Tauri v2 + Rust（后端/原生能力）+ Web 前端（Vue/React 复用现有二开页面）。
- **分发**：企业内网 `.exe`/`.msi`（Windows）和 `.dmg`/`.pkg`（macOS）。

## 范围

### 在范围内

- Windows 10+ / macOS 12+ 桌面应用。
- 安装包预配置服务器 URL。
- 应用级企业邮箱 SSO 登录。
- 原生一级导航框架。
- WebView 嵌入现有 B/S 聊天页面。
- 系统通知、托盘、开机自启动。
- 品牌隐藏与设置适配。

### 不在范围内

- 移动端应用（iOS/Android）。
- 离线消息推送。
- 语音/视频通话原生能力（若 Web 已支持则通过 WebView 呈现）。
- 一次性将所有功能 C/S 化。
