## 详细功能清单

### P0：企业知识库

| 功能 | 描述 | 验收标准 | 推荐方案 |
|------|------|----------|----------|
| 文档创建与编辑 | 支持 Markdown、富文本、代码块等格式。 | 用户可创建、编辑、保存文档。 | BookStack / Wiki.js |
| 权限管理 | 按空间/书架/页面设置读写权限。 | 不同角色看到不同内容。 | BookStack / Wiki.js |
| 全文搜索 | 支持中文分词、标题/内容搜索。 | 搜索结果相关度高，响应快。 | BookStack / Wiki.js |
| 知识同步 | 数字员工可读取用户有权限的知识。 | 数字员工回答基于知识库内容。 | 通过 API 集成 |
| 版本历史 | 文档变更可追溯。 | 可查看历史版本并回滚。 | BookStack / Wiki.js |

### P0：技能分发

| 功能 | 描述 | 验收标准 | 推荐方案 |
|------|------|----------|----------|
| Skill 市场 | 浏览、安装、更新 skills。 | 管理员可从市场安装 skill。 | 自研 + Dify/Coze Studio |
| Skill 注册 | 开发者提交自建 skill。 | 提供 skill 描述、输入输出、权限要求。 | 自研规范 |
| Skill 分配 | 将 skill 分配给岗位/数字员工分身。 | 不同岗位拥有不同技能集。 | 自研 |
| Skill 调用 | 数字员工在对话中自然语言调用 skill。 | 用户无需记住命令。 | 数字员工 runtime |
| Skill 组合 | 支持多个 skills 串行/条件执行。 | 数字员工可完成多步骤任务。 | Dify Workflow / 自研 |

### P1：协作文档

| 功能 | 描述 | 验收标准 | 推荐方案 |
|------|------|----------|----------|
| 在线编辑 | 多人实时或异步编辑文档。 | 编辑内容可保存、可协作。 | OnlyOffice Document Server |
| 文档嵌入 | 文档可嵌入 Teable/IM 对话。 | 用户在 Teable 中可直接查看文档。 | OnlyOffice WOPI/API |
| 文档生成 | 数字员工可创建文档初稿。 | 数字员工调用 skill 生成文档。 | 通过 API 集成 |

### P1：协作日程

| 功能 | 描述 | 验收标准 | 推荐方案 |
|------|------|----------|----------|
| 共享日历 | 个人/团队/资源日历。 | 用户可创建和订阅日历。 | Nextcloud Calendar / Baikal |
| 会议预约 | 查看空闲时间并创建会议。 | 数字员工可帮用户预约会议。 | CalDAV + 自研逻辑 |
| 日程同步 | 与外部客户端同步（iOS/Outlook）。 | 支持 CalDAV 协议。 | Nextcloud Calendar / Baikal |

### P1：协作会议

| 功能 | 描述 | 验收标准 | 推荐方案 |
|------|------|----------|----------|
| 视频会议 | 支持多人音视频、屏幕共享。 | 用户可从 IM/日程一键入会。 | Jitsi Meet |
| 会议录制 | 录制并保存会议。 | 录制文件可回放、可检索。 | Jitsi Meet |
| 会议摘要 | 数字员工生成会议纪要。 | 会后自动输出摘要和待办。 | 数字员工 + ASR/LLM |

### P2：复杂 Skill 编排（长期观察）

| 功能 | 描述 | 验收标准 | 推荐方案 |
|------|------|----------|----------|
| 可视化工作流 | 开发者编排多 skill 工作流。 | 拖拽或 YAML 配置工作流。 | Dify / Coze Studio |
| 定时任务 | 工作流按 schedule 运行。 | 可设置定时触发器。 | Dify / 自研 |
| 异常处理 | 工作流失败可重试/通知。 | 失败有日志和告警。 | Dify / 自研 |
